### Ejercicio 8

Número de letras: Imprima el número de veces que se repite cada letra del abecedario entre todos los elementos de la lista.

```python
#Por cada carácter ASCII de la 'a' a la 'z'
for ascii in range(97, 123):
    # Cogemos la letra correspondiente del bucle y su contador
    letra = chr(ascii)
    contadorletra = 0
    # Recorremos todos los nombres y caracteres
    for nombre in nombres:
        # Recorremos cada carácter del nombre
        for char in nombre:
            # Si el carácter coincide con la letra actual (ignorando mayúsculas y minúsculas), se suma 1 al contador
            if char.lower() == letra:
                contadorletra += 1
    # Imprimimos el resultado por cada carácter
    print("La letra", letra, "aparece", contadorletra, "veces.")
```
