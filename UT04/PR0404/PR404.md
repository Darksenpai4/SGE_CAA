## Ejercicio 4

Supón un diccionario donde cada clave es una asignatura y el valor correspondiente una lista de estudiantes matriculados, tal como se muestra en el diccionario de ejemplo. Crea un programa que tenga un menú con tres opciones:

- Listar estudiantes matriculados en una asignatura
- Matricular un estudiante en una asignatura
- Dar de baja un estudiante de una asignatura.

```python
asignaturas = {
    "Matemáticas": ["Ana", "Carlos", "Luis", "María", "Jorge"],
    "Física": ["Elena", "Luis", "Juan", "Sofía"],
    "Programación": ["Ana", "Carlos", "Sofía", "Jorge", "Pedro"],
    "Historia": ["María", "Juan", "Elena", "Ana"],

    print("\nMenú:")

    print("2. Matricular un estudiante en una asignatura")
    print("3. Dar de baja un estudiante de una asignatura")

    opcion = input("Selecciona una opción (1-3) o 'fin' para salir: ")

    match opcion:
        case "1":
            asignatura = input("Introduzca la asignatura para ver a los estudiantes listados: ")
            if asignatura in asignaturas:
                print("Estudiantes en la asignatura: ")
                for estudiante in asignaturas[asignatura]:
                    print(estudiante)
            else:
                print("La asignatura no existe")
        case "2":
            asignatura = input("Introduzca la asignatura donde quiera matricular: ")
            alumno = input("Introduzca el nombre del alumno a matricular: ")
            if asignatura in asignaturas:
                if alumno in asignaturas[asignatura]:
                    print("El alumno ya está matriculado en la asignatura.")
                else: 
                    asignaturas[asignatura].append(alumno)
                    print("Alumno matriculado correctamente")
            else:
                print("La asignatura no existe")
        case "3":
            asignatura = input("Introduzca la asignatura donde quiera darse da baja: ")
            alumno = input("Introduzca el nombre del alumno a dar de baja: ")
            if asignatura in asignaturas:
                if alumno in asignaturas[asignatura]:
                    asignaturas[asignatura].remove(alumno)
                    print("Alumno dado de baja correctamente")
                else: 
                    print("El alumno no existe en la asignatura")
            else:
                print("La asignatura no existe")
        case "fin":
            break
        case _:
            print("Opción no válida. Por favor, seleccione una opción del 1 al 3 o 'fin' para salir.")
```

## Ejercicio 5

Escribe una función que tome un diccionario y devuelva otro con las claves y valores intercambiados (lo que antes eran valores ahora son claves, y viceversa).

```python
diccionario = {
    1: "milanesa",
    2: "pebete",
    3: "hamburguesa",
    4: "pomelo",
    5: "cafe",
    6: "especia"
}

def intercambiar(diccionario):
    claves = []
    valores = []
    for clave in diccionario:
        claves.append(clave)
        valores.append(diccionario[clave])
    diccioInter = dict(zip(valores, claves))
    return diccioInter

print(intercambiar(diccionario))
```

## Ejercicio 6

Escribe un programa que tome dos diccionarios de productos y precios, y combine los productos comunes sumando sus precios, sin duplicar los elementos únicos.

```python
precios1 = {
    "manzana": 1.5,
    "pera": 2.0,
    "plátano": 0.6,
    "pomelo": 3.4,
    "pintura": 9.0
}

precio2 = {
    "desodorante": 4.5,
    "plátano": 1.6,
    "miel": 5.0,
    "pintura": 11.0,
    "pomelo": 2.5
}

def combinar(lista1, lista2):
    for clave in lista1:
        if clave in lista2:
            lista2[clave] += lista1[clave]
        else: 
            lista2[clave] = lista1[clave]
    return lista2

print(combinar(precios1, precio2))
```
