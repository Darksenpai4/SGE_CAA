## Ejercicio 8

Crea un programa que implemente el clásico juego de piedra, papel, tijeras, lagarto y spock.

Las normas de este juego son:

- Tijeras cortan papel
- Papel tapa piedra
- Piedra aplasta lagarto
- Lagarto envenena a Spock
- Spock rompe tijeras
- Tijeras decapitan lagarto
- Lagarto debora papel
- Papel desautoriza a Spock
- Spock vaporiza piedra
- Y como siempre, piedra aplasta tijeras

Ganará el primero que gane 5 manos.

```python
from random import *

print("¡Juguemos al piedra, papel, tijera, lagarto, Spock!")
manos = { "piedra": ["papel", "Spock"] ,  
         "papel" : ["tijeras", "lagarto"],
         "tijeras": ["piedra", "Spock"],
         "lagarto": ["piedra", "tijeras"],
         "Spock": ["papel", "lagarto"]
}
manosnum = ["piedra", "papel", "tijeras", "lagarto", "Spock"]

ganadas_maq = 0
ganadas_hum = 0

while(ganadas_maq < 5 and ganadas_hum < 5):
    mano_humano = input("Introduzca su mano: ")
    mano_maquina = randint(0, 4)
    maquina = manosnum[mano_maquina]
    print("La máquina escoge: ", maquina)

    if(mano_humano == maquina):
        print("¡Empate!")

    elif(maquina in manos[mano_humano]):
        print("¡Perdiste! Bien jugado")
        ganadas_maq += 1
        
    elif(mano_humano in manos[maquina]):
        print("¡Ganaste! Bien jugado")
        ganadas_hum += 1

    print("Puntuación --> Tú: ", ganadas_hum, " | Máquina: ", ganadas_maq)
       
if(ganadas_maq < ganadas_hum):
    print("¡Has ganado! :)")
else:
    print("¡Has perdido! :(")
```

## Ejercicio 9

Crea un programa que genere los primeros n números de la secuencia de Fibonacci. Recuerda que la secuencia de Fibonacci es:

```
fib(0) = 1
fib(1) = 1
fib(n) = fib(n-2) + fib(n-1)
```

```python
n = int(input("Proponga un número: "))
fibonacci = []
for i in range(n):
    if(i == 0):
        fibonacci.append(1)
    elif(i==1):
        fibonacci.append(1)
    else:
        secuencia = fibonacci[i-1] + fibonacci[i-2]
        fibonacci.append(secuencia)

print(fibonacci)
```
